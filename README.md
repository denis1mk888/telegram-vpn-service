
# Telegram VPN Service

Полнофункциональный VPN-сервис в Telegram с подписками и приёмом оплаты. Построен с нуля, работает в продакшене с реальными пользователями.

## Возможности

- Telegram-бот на aiogram 3.x для управления подписками
- Приём оплаты: Platega (рубли — СБП/карта) и USDT (сеть Polygon) напрямую
- Интеграция с VPN-панелью Marzban: создание пользователей, продление, выдача конфигов
- Система подписок с тарифными планами и автопродлением
- Реферальная программа с авто-начислением бонусов
- Промокоды со сроком действия
- Админ-панель со статистикой: пользователи, оплаты, конверсия, тарифы
- Webhook-сервер на aiohttp для обработки платежей

## Стек

Python, aiogram 3.x, aiohttp, Marzban API, PostgreSQL, Docker, Linux

## Запуск

1. Заполнить конфигурацию (токены, ключи) — см. `.env.example`
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить: `python bot.py`

## Безопасность

Все секретные данные (токены, ключи, доступы) вынесены из репозитория и задаются отдельно.
